# 원격 저장소에 업로드하기

## 1. 원격 저장소에 연결

```bash
$ git remote add origin https://github.com/Bonbon315/TIL
```



## 2. 원격 저장소에 연결됐는지 확인

```bash
$ git remote -v
```



## 3. 맨 처음 원격 저장소에 올리는 경우

```bash
$ git push -u origin master
```



## 4. Git Push

```bash
$ git push
$ git push origin master
```



## 5. Git pull

```bash
$ git pull
$ git pull origin master
```
